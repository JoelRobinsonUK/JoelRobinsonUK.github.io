Todd Howard is Going to be Late

Abstract


This game began as a hasty attempt to hand something in when I lost all the files to my originally planned narrative game. I took a 3D model of my bedroom that I had worked on for another assignment into Unity and added a simple fetch quest. I chose to theme the game around Bethesda's Todd Howard as Fallout 76 had just come out and I had a lot of feelings that needed venting about the whole thing.


This playthrough of the game highlights some of the logarithmic fall-off I experimented with.

Play on itch.io

Context


This game is an escape room style 3D game where the player must collect the character’s possessions from around the room and leave. The game contains a variety of environmental storytelling elements that rely on niche context and humour.
This game’s mechanics are inspired by slow paced narrative games like Gone Home and Everybody's Gone to the Rapture. However, the theme of the game is mocking Todd Howard, Bethesda’s Creative Director and a nexus of internet meme activity.
The environmental assets for this game were created from scratch using Autodesk 3DS Max. Each object was also uniquely textured using Substance Painter. As I had already made these assets, I considered building on the environment to create a large playing space and give the player more to do, however I felt that expanding the area beyond one room did not offer enough opportunity to improve on game mechanics to be worth the time needed to design additional assets.

Technical Stuff


The scripts for this game are very simple. The Item script is attached to every object the player needs to collect, and checks whether the player is in the object’s collider. When the player is in the collider the script displays a GUI element that prompts the player to pick the object up. When the player presses the appropriate key, the object is destroyed, and the relevant Fungus block is executed. The Game Manager script checks that all the objects have been collected before prompting the player to exit the room with another Fungus block and deleting the Game Manager object.
The room environment is surrounded by a reverb zone using the “room” preset, this makes the audio elements played inside the zone sound more like they’re coming from an interior space. The music coming from the desk area is set up with a logarithmic rolloff that means it is audible throughout the whole room but gets quieter further from the source. The music playing behind the door uses a linear roll off that gives it a smaller range.

Reflection


I feel that as a piece of satire this game is very successful, however as a narrative experience it doesn’t meet the standard that I aspire to. I think had there been time to recover my lost original project the narrative I had planned for it would have better suited the requirements. I also feel that the room could benefit from more interactivity beyond the essential items. I do think that this game reaches a high level of visual polish, due to all the models and textures being based off real life objects.
